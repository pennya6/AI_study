# 1.3 애플리케이션 사례

다음은 구체적인 머신러닝 작업의 사례와 그에 따른 기술입니다.
+ 제품이미지를 분석해 자동으로 분류 : <strong>CNN(합성곱 신경망 / Convolutional neural network)</strong> -> 이미지 분류 작업
+ 뇌를 스캔하여 종양 진단 : CNN -> 시맨틱 분할 작업
+ 뉴스기사 분류 : <strong>NLP(자연어 처리 / natural language processing)</strong> -> 텍스트 분류, RNN(순환 신경망 / Recurrent neural network), CNN, 트랜스포터를 사용하여 해결
+ 토론포럼에서 부정적인 코멘트 구분 : NLP -> 텍스트 분류
+ 긴 문서 자동 요약 : NLP -> 텍스트 요약
+ 챗봇, 개인비서 만들기 : <strong>NLU(자연어 이해 / natural language understanding)</strong> -> 질문-대답 모듈을 포함한 여러 NLP 컴포넌트 필요
+ 성능지표를 기반으로 회사의 내년도 수익 예측 : <strong>회귀(Regression), 선형회귀, 다항회귀, 회귀 SVM, 회귀 랜덤포레스트, 인공 신경망 사용가능</strong>
+ 음성명령에 반응하는 앱 : RNN, CNN, 트랜스포머 -> 음성 인식 작업
+ 신용 카드 부정거래 감지 : <strong>이상치 탐지</strong>
+ 구매이력을 기반으로 구분후 마케팅 전략 계획 : <strong>군집(clustering)작업</strong>
+ 고차원의 데이터셋 그래프 표현 : 차원축소기법(dimensionality reduction) -> 데이터 시각화 작업
+ 과거 구매 이력을 기반으로 상품 추천 : 인공신경망 -> 추천시스템 
+ 지능형 게임 봇 만들기 : <strong>RL(강화 학습 / reinforcement)</strong>
